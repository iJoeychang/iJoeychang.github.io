读《潜伏在办公室》

发表于 2017-07-16   |   分类于 读书心得   |     |   次阅读

读这本书，感觉被毁了三观，反思自己以前的行为和表现，出了一身冷汗。

此书一直在宣扬一种观念，“弱肉强食，适者生存，职场上面，处处陷阱，险象环生”。

不知道其它行业是不是这样，感觉程序员的世界比较单纯，或者说很多人不谙世事。

可以说这本书是教人怎样使用权术为自己谋私服务的。

书中说，职场中，你要么是主角要么是龙套，办公室政治、客户的责难、同事的排挤、上司的打压，这些是你经常会遇到的问题。职场有职场的潜规则，不懂一些常识和厚黑学，可能你被卖了还帮人家数钱。要知道人心隔肚皮，知人知面难知心，生活中，职场上，混得好坏，全靠心机和演技。这听起来有些残酷，却大多是事实。

你与同事平等么？你与上司关系好么？你有职场目标么？这些都是书中抛出的问题。书中刻薄地揭露事实，让人不想接受，却又让人无法反驳。

“志向是什么？志向就是你愿意为它而放弃别的东西。人类进步的源泉，是欲望，永不停歇的欲望，你要么胸怀大志，要么赚钱……”
“一个充满野心的人只可以相互利用而不能交心。”
“世界上大多是反射性性格人，容易受人影响，相信别人的观点，缺乏自己的逻辑思维能力。这样的人，只能被奴役。”
“独立思考 这个世界上的一切知识，都是需要甄别的，不是所有名人都是好的，不是所有明言都是对的，不是所有知识都是有用的。”
“诡辩家专用技巧–九真一假。”
“一定要有缺点，不遭人嫉是庸才，常遭人嫉是蠢材。”
“白玉无暇，只会怀璧其罪。”
“不要等别人的恩赐。”
“真正的朋友之间是不会去涉及利益的，他们会刻意地规避这种可能性，只要他们足够聪明。”

电影艋口，黑帮老大说，风往哪个方向吹，草就往哪个方向倒。年轻时候我也觉得自己是风，到处追逐梦想。但被碰的头破血流之后才明白，原来我们每个人都是草。
如果你认识到自己是一棵草，那就可以长出牙齿和棱角，但遇到风的时候，还是要把头低下。
我们做人有时候可以做自己，有时候却必须做别人喜欢的自己。这的确是很大的悲哀，但如果你有自己的目标和信仰，那你必须去做，人要实现目标，是要付出代价的。
千万不能小看一个心怀梦想的人。

书中到处是像以上那样的道理。

开始看这本书，感觉像厚黑学，自己还很抵触，可是读着读着，我就发现自己很像书中列举的某个人物，那么真实！

一本书，我用了差不多四个晚上就读完了，而且几乎没怎么感觉到累，想想，应该是触碰到自己的神经了。书中的事例，让自己深有体会。

这本书，让我心有余悸的是，有些地方触碰到我的痛点了。

回忆自己曾经在工作中口无遮拦说过的话，回忆自己曾经那么相信、信任的人最后却让我很失望，我陷入了沉默。玩心机的人，八卦的人，很是让人厌，可是如果你不这样，你太单纯了，到最后你会发现吃亏的、受到伤害的一定是你，你得不到好的同时，还会被人骂到蠢，这就是现实。

工作中，私下里，你说的每句话，老板都会知道，告密者可恨，耳报神可恨，你感觉自己被出卖了，你咬牙切齿，可是认真反思反思，你应该意识到，其实错的是你，你不应该太单纯。职场中，说话要谨慎，有些话不能说，有些话不能对某些人说。

这是长大后的悲伤，童年时的天真无邪让人留念，可是如果长大后你再天真，那就真的是幼稚了。

你说别人坏话太多，聪明人会觉得在背后你照样会说他。

有些朋友的关系变淡、生疏、甚至断交，有时可能仅仅是因为你获知对方在别处怎样怎样说你的坏话。

虽然在江湖上行走，“何人不被说，何人不说人”，但是如果是朋友的话，背地里还是不要议论他人是非的好。说是非者必是是非人。

对于职场上混，作者给了些建议，诸如：

一、小心驶得万年船
  1. 不该听的听不到，不该说的不说
  2. 该你做的事情做，不该你做的事情不做
  3. 要摸清公司的脉络
  4. 要选择朋友
二、前期不要急

不要急
沉下心学习，成功没有捷径，就是靠积累靠关系靠运气。欲速则不达。

不要怕
上司可以掌控你的职场命运，但他不应该掌控你的情绪和理想，更不能掌控你的思想。

更自私
自己的利益放在集体利益之前，自己的理想放在公司理想之前。

跳槽前问自己几个问题
职务有没有升，薪水有没有增加，公司有没有前途，是否符合你的职场目标。

权术
真正的术是没有术，无招胜有招，大巧不工，大智若愚。

三、一点心理共振

城市里最多见的一种人，平平安安的读书，安安稳稳的生活，但是职场的残酷却改变了他们，让他们不得不放下从前的温顺，为自己组装到牙齿。

在这个世界上，有人害你，就一定有人爱着你。有人嫉妒你，就一定有人看好你。有人厌憎你，就一定有人喜悦你。
人生本就有得有失，你走得快，收获就会比失去多，而停下来，那些妖魔鬼怪反而蜂拥直上。
不用理会，你只管往前走，渡过他们、经历他们、超越他们…然后用远去的背影表达你的态度。
有些人，是注定遗世独立的。

贪欲、性欲、野心这些都是人性中最实实在在的东西，但现实中，我们却要把它们都隐藏起来，甚至装作没有。难道这不是虚伪和谎言么？
我们不是在说谎的过程里，就是活在谎言里。

有些人喜欢在同事聚会时骂公司骂老板，然而这些话，总会在必要的时候，传到 BOSS 的耳中。与同事相处时，放下工作上的烦恼，表现出生活里的另一面，这种豁达，能够让人将你引为知己。

与这本书接下缘分，还是因为前辈的推荐。读完此书，感觉太好了。我想这本书的观点会对我之后的为人处世产生很大的影响。实在是一本好书。读完此书，缘分自此结下。

我不想害人，也没有害人的心，我只想保护好自己不被人伤害。我会学术，包括权术、计谋、心计，但是我永远不会拿术来对待朋友、亲人。战国时期的鬼谷子，他是玩弄权术、阴谋、阳谋的大家，他以一人之力将诸侯列国玩弄于鼓掌，他的成就、他的学术造诣，让人佩服、让人惊叹。我想向他学习，但希望自己永远不会失去向善的心。
