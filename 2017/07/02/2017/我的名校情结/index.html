我的名校情结

发表于 2017-07-02   |   分类于 我的随笔   |     |   次阅读

前两天有个亲戚问我当年高考报考的事情，想想我都毕业五年了，高考离我已经是很久远的事情，这不免勾起了一些我对往事的追忆。

那位亲戚他妻弟刚刚高考结束，理科生，考了有 550 分，今年河南一本线应该四百八十多，他那成绩应该算很好的了。记得08年我考了546，差几分还不够一本线，而十年后，分数线都变这么低了！不知道是不是因为题太难，还是因为扩招太厉害，为此我特意查了下名校的录取线，国内一线学校的录取成绩几乎没怎么变，在河南还是非要六百多才能上985院校。看来一般学校和名校差别在越来越大。

河南这么大的省，这么多的人（总人口占全国差不多十分之一），全省竟然没有一所985院校，本省最好的大学郑州大学，录取分数非常高，可是出了河南和外面很多学校比起来还是没什么竞争力。想想也算是河南的一个悲哀吧。河南各方面都在发展，也有不错的成绩，可是教育却远远落后于同等级别的省份。我在想，如果可以的话，我要让我的下一代走出河南。如果条件允许，我会让他去国外留学几年。当然这些现在还只能是想象。

其实我的名校情结很严重，只可惜当年我没有能够遂愿，至今回想，还是我的一大遗恨。可是时间已经过去，再怎么追悔都已经于事无补，好在东隅已逝，桑榆非晚，现在努力些，想必努力后未来也不会差了。有时文凭占一项，但它只代表过去，未来的赛跑还长着，结局谁胜还不好说。现在已经是终身学习的时代，无论以前你怎样厉害，终止学习，你肯定是要被时代所淘汰的。

这两天读了点吴军博士的得到专栏，其中有个大学学什么和在二流大学如何学习的话题。他总结为三点，我摘抄如下。

一、生活最重要

大学里，学习是第一任务，但人生的任何阶段，都只有一个根本，就是“生活”。
学习只是生活的一部分，而不是全部。
18-22岁左右是人生最美好的时光，如果都泡在教室、图书馆和宿舍里，实在可惜。这个时间谈一次恋爱，对人的一生都有好处。
在大学里爱一次，同时享受被爱一次，实在是人生不可缺少的经历，错过了，将来可能不再有，而没有它，人生则不完整。
人生最浪漫而美好的经历应该是那些美丽的邂逅。当你从图书馆的楼梯走下，迎面遇到一个匆匆跑去上自习或者还书的女生（男生），不经意之间两个人互相看了一眼，批次内心一笑，然后开始一段情缘。这种美感是人生其他任何阶段都难以经历的幸福。

二、 交一些挚友，每个人一生的朋友。

在大学里很容易考察出一个人能否成为你的挚友，这是工作十年也做不到的。年轻人一起生活学习四年，彼此是坦诚的，完全有机会了解对方的内心和性格上的每一个细节，但是在工作中交往时间很长的人，彼此不一定能做到这一点。
大学生通常功利心不会太强，彼此抄个作业或者帮个什么别的忙不是什么大事，也不会因此决定欠对方一个人情。但是一旦走出校门，大部分人在“你的”和“我的”之间就分得特别清楚。老板安排下任务来，你需要别人帮忙，对方哪怕帮了你一点点，你就欠一个人情，会对他很客气地表示感谢，下一次他对你也是如此。在这样彼此客气的环境中，人是很难深交的。

三、尽可能多地参加一些和学习科研无关的社会活动，这是成为社会人的必经之路。

广义上的学习和狭义上的学习。前者包括学做人、思想上的修行等等，后者特指知识、技能和方法方面的学习。
为了生活而学习，而不是为了专业而学习。
大学学什么：

  1. 付诸行动的习惯。
  2. 沟通能力和表达能力。
  3. 表示友善和爱的能力。
  4. 总结、写作和逻辑推理的能力。
    只有善于写作和记录，总结第一次成功的经验，或者某一次失败的教训，才能让自己做第二件事的时候站在第一次的基础之上，第三次站在第二次的基础之上，这样才能叠加效应，才能不断前进。

看了他的建议，回想自己的大学时光，感觉有好多都可惜了。那个时候，可能我最需要的是有个高人给指点一下，包括心理方面的、学习技巧方面的、为人处事方面的。如果我处理得当，应该不会出现之前发生的很多尴尬的事情，可能无论学业、工作、生活方面，我都应该比现在强。只是往事只能回味，不能修改，我只
能寄希望于将来了。

现在我会刻意去学一些心理学和经济学相关的知识。这两科是我大学期间就想研究的方向，因为种种原因未能系统地学习。现在要学，不能再找任何借口不学。

未来我会在在线教育这个领域做点事情，希望可以给教育不公平待遇下的人们一点帮助。
另外，学习是一辈子的事情，我会学到老。十步之内，必有芳草。十室之邑，必有忠信。要向所有比自己强的人学习！
