写在南方小年

发表于 2017-01-21   |   分类于 我的随笔   |     |   次阅读

昨天祭灶节，又正值是二十四节气的大寒，今天是南方小年，眼看马上就要新年来到了，此刻我正坐在理发店，准备修修头发，新年新气象，一起从头开始。

昨晚闲来无事，兴许因为一个人孤单无聊，有那么一会儿心情不是太好，不过那样的状态很快就过去了，毕竟美好的事情还是很多的。

晚上八点多些，给家里打了电话，和三弟聊了半小时。家里的事情感觉还是很多的，让人有些牵挂。以前很多时候我有会嫌两个弟弟能耐浅，有些替他们前景担忧。现在觉得，他们也在慢慢成熟，我的担忧和烦恼，可能有些偏多了。各人各有各人福，借用一位亲戚的话，”小鸡还会挠个食儿呢”，无论谁，未来自己还得靠自己。兄弟间，我能帮衬些，但不可能完全靠我一人，我也会力不从心。以前总担心我三弟这个小家伙会不走正途，现在想，可能我有些过分担忧了。

心情平静后，坐在书桌前，想着要做点什么事情才好呢，不然时间虚度的话，会觉得是罪过。尼采说，“每个不曾起舞的日子,都是对生命的一种辜负”。想抓住时间做些有意义事，只是又不想看专业相关的东西，于是看着桌子上堆积的老高的书，随手拿来一本小书翻开开始看。

那本儿书是鬼脚七的新书，双11搞活动淘来的。本来我是有些不屑看这种类似的书的，会感觉是在浪费时间，因为平常可以拿来消费的业余时间真的太少了，而又有那么多事情要做，很多兴趣都想去培养，已经忍痛割爱了很多。如果再花费时间在没有太多营养的书籍上面，会觉得不值得。平常我喜欢读古书，喜欢读经典，一直觉得只有能够经历过时间考验的书籍才是值得深读的，而时下流行的那些所谓畅销书，可能只是仅仅是流行而已，存活时间不会太久。

当时买鬼脚七的纸质书，是因为经常读他的公号文章，觉得他写得好，算是对他的一点支持。再者此书是记录他不带钱徒步几千里从五台山到峨眉山的出家过程，个人感觉那种体验很酷。说实在的，我也很想体验一把。

因为鬼脚七文章影响的缘故，最近一年，我抄了很多遍佛经，想着有一天，我要把它们送到南海，想为先人祈祷。

有时会想，这个世界有太多的未知，我们是谁？我们从哪里来？我们又要到哪里去？这些曾经困扰过，现在困扰着无数人的简单而又终极的问题，科学无法做出解释。这个世界，到底有没有鬼神，谁也证明不了，起码现在是这样。西方国家，科技那么发达，其国民十有八九是信耶稣的，就连牛顿，老年也皈依了圣教，一直在研究神学。

我不信任何教派，但潜意识里还是希望有鬼神的存在的，因为如果真可以那样，百年以后还会有灵魂在，想想也是不错的。

因为感受到佛经的深奥、经典、微言大义，我有被吸引，抄了些佛经，看了些相关书籍，电影看了部玄奘大师传，电视看了惠能六祖传，深为大师们的学识、人品、修为所震撼。我想即使我不信神佛，但是佛陀的智慧还是可以深入学习的。

现实中，跑步，练字，写作，甚至是烹饪，我都觉得是自我的修行。我写作，我练字，我跑步，我在修行。

只是目前我道行太浅，写作水平还太有限，字还写得太难看不敢示人，跑步还没能持续坚持，这些都是不足。

昨晚，无意间又看到禅宗六祖惠能大师的禅偈，抄了一遍，拍下来想晒到朋友圈，感觉不好看，就又抄了一遍，还是不好看，又抄了一遍，反反复复抄了五六遍，终于没敢示人。字太丑了。于是，我发愿，此生一定要把字写好。

惠能大师可以称为是一个传奇，他不识字，却写了一部坛经，是唯一一部中国人写的佛经。那首著名的“本来无一物，何处惹尘埃”也是他说的，那个“是风动是幡动还是心动”的辩证例子也是源自坛经里面，孔子、老子和他分别代表中国的儒道释，其成就之大可见一斑。昨晚我抄写的那首他的禅偈，这里共享一下：

心平何劳持戒，行直何用修禅。

恩则孝养父母，义则上下相怜。

让则尊卑和睦，忍则众恶无喧。

若能钻木取火，淤泥定生红莲。

苦口的是良药，逆耳必是忠言。

改过必生智慧，护短心内非贤。

日用常行饶益，成道非由施钱。

菩提只向心觅，何劳向外求玄。

听说依此修行，西方只在目前。

-- 2017.01.21 侵夕
        上海 新泾北苑
