新办公室

发表于 2017-05-11   |   分类于 我的随笔   |     |   次阅读

今天是在新办公室办工的第一天，感触颇多。

想想在春秋就要三年半了。三年多的时间，说快很快，快到自己觉得过得颓废，时间消失了，自己却很不满意自己取得的进步或者说成绩；说慢也慢，毕竟是三年多，人的一辈子能有多少个三年多呢?

昨晚搬家整理东西的时候，翻看放在桌子上面很久很久的几本薄笔记，纸张旧旧的都变了颜色，里面随便记录的是自己以前开发时所画的逻辑，字体大多潦草，看着那些字，让我很是不满意自己的字体，心想以后我必须下大工夫把字练好。拿着那些没有用处的本本，大多翻看几下就随手丢掉了。每次搬家都会扔东西，这次当然不例外。只是在一个本子里面看到一页内容，写的是2015年计划清单。那应该是我在春秋第一年年末写的东西。看着是2015年的清单，看着里面的内容，我顿时感到羞愧极了。列出来的那几件任务，至今我都没有完成，而现在2017年都快要过半了。那几件事，有的是没有能力完成，而有的则是没有实践去完成。想想规划不能实现，大多不是不能为，而是不为。

在新的办公楼里面，环境感觉好了很多，办公室不再是一个屋子一下容纳几百人那样的大办公区，同时座位也重新分配了下，就我而言，是比较满意的，你知道，有时座位不爽，那是极影响心情的事情。

再有就是新办公楼是新楼，大厅以及楼层都装饰得略显豪华，给人感觉有些高大上，起码相对而言是这样的。不爽的是，现在吃饭没有以前在自己餐厅那样方便，也没有以前那样好吃，而且新大楼里的物业公司服务态度很不好，给人感觉不爽。这个事吧，可能也是相对的，那些物业人员看着这么多不太配合他们的来来往往的人员，可能他们也会感觉很不爽。

最后一点，就是自己充分感觉到了人际关系的复杂性和重要性。身为技术人员，一心闷头工作，本来也无可厚非，只是与人相处，让对方舒服，也让自己舒服，这样的技巧还是需要学习的。比如你看一个人不爽，那没事，看两个人不爽，也没事，但是看所有人不爽，那就是自己的问题了。朋友曾说，“你看别人不爽，别人还看你不爽呢！” 我想是的，此话言之有理。

想想自己一两年前做的事情，真的只能说是尴尬人做的尴尬事吧。起初从 IT 搬到电商的时候，那时我极不情愿，现在又从电商搬到 IT ，我又很不情愿，究其原因，无非是因为相同的人和事。至今时间过去了很久，自己也算看开和顿悟了，某些人和事，都不会再像以前那样干扰和影响我了。只是，话虽如此说，见到他们，想起以前事，感觉还是蛮尴尬的。

曾经不止两三个人说我情商低，开始我还不服气，现在看来，他们或许是对的。今后，生活这本书，以及人与人之间的诸多事，我还是要虚心好好学习才好。

现在的这一年，从天气转暖开始，我逐渐总结了一些适合自己的学习、工作方法，也养成了一些诸如早睡、早起、坚持锻炼的好习惯，希望在新的办公室里，接下来的一年里自己能够在出色完成工作任务的同时也能在技术上有新的突破。
