《摔跤吧，爸爸》观后感

发表于 2017-05-20   |   分类于 影视评   |     |   次阅读

昨晚和三个同事一起去看了这部电影。

本来想周末在网上下载了视频再看的，昨晚临下班，有个同事问要不要一起去看电影，我问看哪个，他说《摔跤吧，爸爸》，我随即说去。我是冲着阿米尔汗去的。

电影一波三折，可谓感人得很。当影片中女主角获胜时电影播放慢镜头，我旁边的同事说，“我激动得都想吼出来了…”

那个时候，我也很激动，我想应该有很多人会哭的。

第一次看阿米尔汗的电影还是在我大一时候。记得那时候精神状态不佳。一次课堂上，有位贾老师推荐了电影三傻大闹宝莱坞，他说那电影很好看，还说生病了心情不好，看了那电影就什么都好了。

开始我不以为意，心想不就一部电影么，有什么好的！还是有一天在宿舍里闲来无事，我想找一部电影看，然而又不知道看什么好，那时想起老师的推荐来，觉得可以看看三傻大闹宝莱坞这部电影。

结果看完后当然感觉很不错，三个多小时的时长，竟然不显得冗长。

那以后也就放那儿了。

直到不久前看到有部新电影被评分九点多，心想是什么神剧啊会评分这么高。然后了解点相关知识，才再次关注起来阿米尔汗，这位印度国宝级的人物。

上个周末，用手机在西湖旁边看完了他的电影我的神啊，让我本来不快的心情为之一振。他给人积极向上的思考，那感觉太好了。让我又想要拼斗起来。

本来我是想下载《摔跤吧，爸爸》的，无奈手机上找不到资源，这才下载他的其它电影。不过没有白下，他的其它电影同样好看，我想我还要再看他的其它电影。

国外演员，我最喜欢汤姆汉克斯，其他演员大都不感冒，现在又多了一位感冒的，就是阿米尔汗了。

汤姆汉克斯号称好莱坞一哥，曾主演过多部经典大片，我觉得阿米尔汗给我的感觉和一哥给我的有点像，他们通过影片向人布道积极、乐观、悲天悯人以及真善美。

电影中，阿米尔汗培养他的两个女儿成了两个世界冠军，一家之中能出来一位世界冠军就了不得了，他家竟然出了两位！而且还是在条件极其艰苦的农村。

让女儿们练习摔跤，让她们和男娃子摔，逼她们早晨五点便开始魔鬼般的训练，将自己的梦想强加于孩子身上，这些不近人情的背后，藏着的也是父爱。

人言可畏，尤其在乡下，那种感觉我是怕了的。冠军们的父亲却不理任何人的嘲笑和闲言碎语，这点很难能可贵。

是的，如果总是在乎别人的看法，总是去讨好别人的认可，这样最后只能是成为普通的众人。人应该敢于担当，敢于天下先，敢于冒天下之大不韪。

自己认可的事，就要坚持到底。这期间会有很多人出来指责你，阻挠你，嘲笑你，甚至侮辱你，辱骂你，但是只要你认为自己的坚持没有错，就坚持下去。早晚拿结果给他们看。

当然结果不是最重要的。成功后，固然可喜，失败了，也要坦然处之。
