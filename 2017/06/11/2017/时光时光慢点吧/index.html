时光时光慢点吧

发表于 2017-06-11   |   分类于 我的随笔   |     |   次阅读

傍晚时分，跑步五千米。最近发现了个不错的跑道，以后周末应该会经常去那里跑步，而且再也不能以红绿灯为由说影响跑步成绩了。那里行人很少，根本没有机动车。跑道唯一不好的地方是它紧临着外环，只有一道天然屏障相隔，噪音太大。

今天跑步完毕，在路边树林里听了鸟叫，呼吸了新鲜空气，然后冥思，那感觉很不错。当时我在想，如果可以经常性的在树林里呼吸呼吸新鲜空气，那肯定非常有益健康。另外在安静的树林中进行冥思、写作，那感觉肯定更好。

树林旁边有条小河，天将黑的时候，无意来到了河边，看到有几个人在钓鱼，精心拍了几张照片，效果差强人意，然后想到需要好好学学摄影和图像后期处理，以后那肯定非常有用。我想现在我的水平可以算作刚入门，但以后我肯定会在那个领域做到非常出色，因为有兴趣和持久学习的耐心。我期望我的作品能够让无数人看到。

平常时候太忙，主要忙于工作和学习，然后生活中就少了很多的闲散乐趣。今天在河边，我突然想，为什么我不能钓鱼呢？那样不单可以让时间慢下来，还可以陶冶自己的情操，可以锻炼自己的意志。我想这个想法是好的，也是可以去实践的。

下午午睡了半个小时，起来后搞了个大扫除。然后想着可以读会儿《后汉书》。现在我平均每周都会给二十四史一个晚上的阅读时间，因为这周三没有顾得上读，周末就想着要补过来了。下午四五点间，当西边洒来的阳光透过窗帘投到我的书桌，我看着那阳光，一边听着音乐，一边读着古文，偶尔再呷一口安吉白茶，时光很慢，我很享受。虽然书读了几十页，发现读懂并记下的不多，但那种感觉还是不错，你知道，是那种 feel ，很充实的感觉。

今天本来只想读两个番茄时钟，也就是差不多一个小时，读完预订计划，再看时候已经过了两个小时了，时间超了，但是还差一点没有读完规定的内容。读完书，天已侵夕，再去外面跑五千米，出了一身汗，感觉不错。

读书，跑步，现在对我而言，是工作之外必不可少的生活内容。今年的跑步目标，现在也差不多完成了一半，很有成就感，不出意外，今年会超额完成跑步目标。至于读书计划，这里不再详谈。

因为工作的原因，因为兴趣的原因，很长一段时间，我与朋友们的沟通很少，然后陷入了一种我执状态，即“自以为是”，“认为自己的都是对的”，“对技术太着迷和执着”，“对时下流行文化看不惯，不接受”，“太着迷于文言”。这样其实不好，时间长了，容易与时下社会脱轨，甚至变为一个怪人。

前不久，一位读我文字的前辈给我指点，说我应该改改知识体系，我觉得蛮有道理的。虽然我骨子里是很难接受别人意见的，即使表面同意，内心也不一定认可，但是那位前辈的说法和指点我还是乐意接受并会去实践的。有时，人真是不是你想怎样就怎样，作为社会群体之一员，你还要学会与更多的人接触打交道。与人相处，这个学问就大了去了。自前辈指教后，我试着读了点行为心理学相关的书籍，感觉受益匪浅。以前总感觉白话文废话连篇，太啰嗦，太无味，现在倒觉得，白话里面同样有很多经典。需要读的书，需要学习的知识，简直如汪洋大海，我们怎么学也学不完的，但我们可以尽自己所能，能学多少是多少。而与人交往，以及人性相关的知识，也是成体系的，有很多知识确实是处事时行之有效的技巧。

平常忙于这儿，忙于那儿，感觉时间真的不够用，大概两年之前，才发现有番茄工作法这样的东西，知道了 GTD（Getting Things Done），然后用了一年的 doit.im 付费会员，感觉很好，每天都安排得满满的，每天都有条有理。一直很懊恼怎么很久以前不知道有这种东西，如果上学时候使用这样时间管理方式，可能我会成绩好很多。好吧，又意淫了。

这几天会员时间到期，因为年费一百多，我突然不想继续付钱了，虽然用着感觉不错，但我翻来翻 app store，以及在 twitter 上看别人推荐，我换了个软件，things 3 。虽然同样付费，但相对便宜了些，而且不再是会员制，是买了之后可以永久用。使用了有几天，感觉体验还不错。

因为这个事情，触到了我的痛点。因为我也是开发软件的，自己最想用的app，竟然是别人开发的，还要给他们付费，这简直是耻辱啊！我想未来我是要做出个类似的软件的。

合理的时间管理，可以更多的做自己喜欢做的事情，可以更好的利用时间，这样可以说就是在改变生活质量。

现在有些好习惯，比如读书、写字、旅行、摄影、跑步、练字、早睡、早起，我在一直经营，并会长期坚持下去。有些坏习惯，比如嗜酒、寡言少语、技术宅，我也会努力改善。好习惯要一直坚持，坏毛病，也要不吝丢弃。

好习惯当然会有不错的回报的。比如写字，因为这个，我多认识了不少的人，其中还有能够给予我不错指教的前辈，感觉一切都是缘分。

因为写博，曾经也有朋友让帮忙写文章，写了一次，感觉很不好，除了有华丽词语的堆砌，很少能读到感情。自那后，我就轻易不敢再接类似的“活儿”了。毕竟我不是专业卖文的，我现在写的大多算是随笔散文？其它的倒还真写不来。不过我还真想有天自己可以写出不错的小说来。

对于朋友的求助，如果我能够写得不错，当然我是很乐意代劳的。有待继续加油了。
