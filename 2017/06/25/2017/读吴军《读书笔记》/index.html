读《吴军读书笔记》

发表于 2017-06-25   |   分类于 读书心得   |     |   次阅读

无意间在得到读了点吴军谈读书。觉得有所收益，现总结一下。
吴博士把他的读书目的分为三部分，即长智、怡情、养性。
而读书方法为：吸收（input）+ 消化 (process) + 产出 (output)。
读书长智的目的就不用多说了，无论阅读什么类型的书，都是为了从中吸收这方面或者那方面的知识。
下面说说怡情和养性。

怡情

每个人享受的方式不同，这里面没有哪一种就是高雅，哪一种就是低俗的分别。有的人喜欢感官上的享受，有的人喜欢精神上的愉悦。无论喜欢什么，都是个人的追求，不分高等，低贱。就像从事不同劳动的工作，在不违法乱纪的情况下，付出的劳动都应该得到尊重。
阅读的好处是可以怡情，不是为了取悦于谁，而是自我享受。
试想在雨天的一个午后，一个人坐在教室的窗前，听着窗外的风雨，读两三个小时的书，那真是一种享受！
年少时，家境不宽裕，买书于我是一种奢望，犹记得儿时一次奢侈行为，是买了一本书《格林童话》。
现在条件好了，却买书少了，已经没有儿时那会儿抚摸书本纸张时那种欣喜感觉。现在买书，很多时候更倾向于购买技术书籍。读技术，是为了增长技能赚更多的钱。有些功利。
而读书不能太功利，应该学会借书怡情。
想想辛弃疾的词，“那更消、几番风雨，匆匆春又归去。惜春长怕花开早，何况落红无数。春且住，见说道、天涯芳草无归路。”
意境多美好啊！中国古典如果没有那许多美好诗词，那中国文化一定残缺很多美。
至于如何成长，把自己变成一个有趣的人非常重要。

养性

吴博士说，对他修养有益的书，重要的有五本：
《曾文正公全集》
尼采的书
《富兰克林自传》
《培根随笔》
《庄子》

每个人都会有自己喜欢的书，每个人都会有自己的哈姆雷特。我还没有读完上面的书，但以后应该会更多得去读的。对博士产生深远影响的书籍，肯定有其独特价值。
读书可以养性，吴博士说，

如果我们把目光放得长远些，生意上的意义要远比文明的成就小得多。赚到一笔钱，成功办了一个公司的人大有人在，缺了张三，还有李四。但是，如果做一个产品，最终对世界有点帮助，这个意义就比较大了。

我们常常讲底层逆袭很困难，其实身处底层，贫穷固然是一个最大的上升障碍，但是更可怕的是深处底层后见识缺乏，而且自制力缺乏。

对于每一个人，养性这件事都可以通过读书来达到。在中国，很多学习和自我提升需要不少的资源和资金，不是所有人都有条件做到。而书籍相对来说，非常便宜。也就是说，通过图书养性，人们很容易获得不受他们家庭财富影响的品格上的财富。

吴军博士已经出了好几本畅销书籍，在国内科技圈影响挺大。读了他的读书笔记，对我的读书方式有所帮助。
现在我不再简单的追求读书数量，不再囫囵吞枣地浏览，而是遇到好书就去认真地钻研，要把书吃透，把书读薄再读厚。最重要的，要有产出。只有产出了，知识宝藏才会成为自己的。
还有一点，现在我把有用的东西都往印象笔记里面丢。时间长了，印象笔记将会是我的储藏宝库。
