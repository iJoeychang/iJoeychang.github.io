我的参悟

发表于 2017-06-01   |   分类于 我的随笔   |     |   次阅读

前几天因为宿舍调换的事情，闹得好多天心情不畅。

我发现我的心理素质不行，遇到事情，总是绕不开，有时纠结着纠结着，能把人愁出病来。以前，因为心理素质不好吃了太大的亏。如果心理素质好的话，估计上学那会儿我不会走那许多弯路了。当时为了绕不去的情节，自己浪费了太多的时间和精力，最坏的时候，整个人还差点崩溃。

加上被人说，最近我开始意识这个问题对我很严重，所以近来我会抽时间学习一些心理学以及如何改善人际交往相关的东西。以前没有投入时间和精力去学习，以前也没有太多的实践练习，现在总是要补回来的，欠的债总是要还的。

因为心情不佳，前天我在手机上删除了微博。刚开始很不适应，因为我发微博比发微信多，而且微博更真我。当时因为觉得刷微博刷微信浪费时间，又觉得把想法暴露出去给人知，感觉怪怪的，加之可以使用Twitter，就删掉了微博。想要克制，也不怎么发微信朋友圈了。后来感觉像少了什么，就在 iPad 上面装了个微博，一天估计只有晚上才会浏览下微博内容，主要是为了看朋友动态。其实之前有想着再新搞个微博账号呢，但是不舍得已有好友，也就不折腾了。

因为状态不佳，学习起来，效果自然不太好。这让我想起以前上学的时候，感觉学习很好的同学，他们大都也很会玩儿。应该是张弛有度，能玩儿才能学，不然就成了”All work and no paly , make jack a dull boy”。

在心情不佳的晚上，我喜欢一个人跑步。有时跑得近，有两三千米，有时跑得远，有近十千米。跑完后，热汗淋漓，小风吹来，那种感觉无以言喻，真的好舒服。当我跑步的时候，以前还会听音乐，现在不听了。跑步的时候，我什么也不做，就那样任身体劳累着，心里什么也不想。算来这也是一种禅修。

说到禅，这一年我读了好几本佛学相关的书，还经常在周末的夜晚或者午后用毛笔抄写佛经，我想等把三部佛经各抄完十遍后就带着自己的墨迹去南海普陀寺拜见观音，主要目的是把我的抄经存到寺中为先人祈福。

我不相信有鬼神，却希望有鬼神。我不信佛，却在学佛。

夜晚，有时我会进入冥想的状态，我打坐，我神游太虚，我总结，我反思，我觉得我自私，我自以为是，我喜欢听好听话，我讨厌逆耳忠言，我发现我有很多毛病，我禁不起诱惑，我想如果让我做官我肯定是个坏官，所以这辈子不做官那是最好不过了。

自从告别了校园进入了社会，因工作故，入了 IT 这行，有时会觉得 “一入 IT 误终身”。觉得如果不是从事这个行业，可能我会有更多的时间去做自己更喜欢的事情，比如经营自己的兴趣，那样可能一个不小心还会让我在某个领域做出一点成绩来。

现在，日子过得飞快，岁月如白驹过隙，我时常会想，“一生就这样过了么？！”

不，当然不能！我不能为技术付出一生。世界上，除了它，还有很多美好的事物。

想起自己在老家时候，我一天可以做很多事情。那一天我去万隆街上卖了一车花生，回来后尚有大半天悠闲时光，可是在沪一天，能解决，或者还没有解决一个问题，已然到了晚上八九点了。

充实点好，但也不能在快节奏中迷失自我。我常想，活着的意义和追求是什么？我时常冥想遐思，终于参悟不透。

昨晚做了一个梦，醒来后梦境忘记了大半，但隐约中经历了这样的场景–

云青青兮欲雨，水澹澹兮生烟。
列缺霹雳，丘峦崩摧。
洞天石扉，訇然中开。
青冥浩荡不见底，日月照耀金银台。
霓为衣兮风为马，云之君兮纷纷而来下。
虎鼓瑟兮鸾回车，仙之人兮列如麻。
忽魂悸以魄动，恍惊起而长嗟。

我竟然好像经历了诗仙经历过的梦境。估计是日有所思夜有所梦了。
